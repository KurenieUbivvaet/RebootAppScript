Скрипт для перезапуска программы
================================
Этот скрипт позволяет перезапустить программу (сервис, службу) в определённое время.

Установка
---------
1. Для начала нам необходимо развернуть папку на компьюте
   - можно скачать архив и разархевировать в любое место
   - можно установить через Git
     1. Для начала в папке, куда хотим добавить наш проект, открываем Git Bush.
     2. Далее пишем `git init` -> после инициализации -> `git clone https://github.com/KurenieUbivvaet/RebootAppScript.git`
2. Далее требуется запустить `script_start.bat` !!ВАЖНО ЗАПУСКАТЬ С ПРАВАМИ АДМИНИСТРАТОРА!!
   * можно можно поставить на автозагрузку

config.json - что для чего надо
-------------------------------
1. `target_time` - указывается время в которое в которое запустится скрипт, в формате `"09:10"`
2. `process_name` - указывается программа уоторую нужно завершить.
   * посмотреть название можно через диспетчер задач, кликнув по программе правой кнопкой мыши и открыть свойства.
3. `file_to_run` - указывается полный путь до файла запуска программы которую нужно запустить.
4. `timeout` - указывается время, в секундах, для паузы между закрытием и запуском программы

все параметры __КРОМЕ__ timeout прописываются в `"кавычках"`, сам timeout обычным целым числом
