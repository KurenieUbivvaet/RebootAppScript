Скрипт для перезапуска программы
================================
Этот скрипт позволяет перезапустить программу (сервис, службу) в определённое время.

Установка
---------
1. Для начала нам необходимо развернуть папку на компьюте
   - можно скачать архив и разархевировать в любое место
   - можно установить через Git
     1. Для начала в папке, куда хотим добавить наш проект, открываем Git Bush.
     2. Далее пишем `git init` -> после инициализации -> `git clone https://github.com/KurenieUbivvaet/RebootAppScript.git`
2. Далее требуется запустить `script_start.bat` !!ВАЖНО ЗАПУСКАТЬ С ПРАВАМИ АДМИНИСТРАТОРА!!
   * можно можно поставить на автозагрузку

Добавляем скрипт в планировщик задач на Windows
===============================================
Откройте Планировщик заданий.
-----------------------------
* Нажмите клавиши `Win + R`, введите команду `taskschd.msc`.

Создайте новую задачу.
----------------------
* В правой панели нажмите «Создать задачу...» (Create Task...).

Вкладка «Общие» (General):
--------------------------
* Укажите Имя для задачи, например, `Запуск моего скрипта`.
* Самое главное: в самом низу окна отметьте галочку «Выполнять с наивысшими правами» (Run with highest privileges).
* Выберите параметр «Выполнять для всех пользователей» (Configure for: Windows 10/11).

Вкладка «Триггеры» (Triggers):
------------------------------
* Нажмите «Создать» (New).
* В поле «Начать задачу» выберите «При запуске компьютера» (At startup). Это обеспечит запуск до появления интерфейса и UAC.

Вкладка «Действия» (Actions):
-----------------------------
* Нажмите «Создать» (New).
* В поле «Действие» оставьте «Запуск программы» (Start a program).
* В поле «Программа или сценарий» укажите путь к вашему файлу.

Сохраняем настройки:
--------------------
Нажмите OK для сохранения задачи. Система может попросить пароль администратора (если вы не под ним), но сам скрипт при старте системы запрашивать ничего не будет.
